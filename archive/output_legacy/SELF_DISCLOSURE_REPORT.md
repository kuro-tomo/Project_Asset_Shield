## 1. 疑惑ポイント: 総リターン 8,646% は現実的か？

### 疑惑
- 18年間で87倍のリターンは「出来過ぎ」ではないか
- 年率29.64%は持続可能か

### 回答・エビデンス

**正当性の根拠:**
1. **期間特性**: 2008年リーマンショック底値からのスタートは構造的に有利
2. **日本バリュー株の特殊性**: PBR<1.0銘柄は日本市場に多数存在（東証上場の約40%）
3. **複利効果**: 18年間の複利は小さな年率差を大きな最終差に拡大

**限界の開示:**
1. **生存者バイアス**: 上場廃止銘柄の完全なトラッキングは未実施
2. **配当未考慮**: 配当再投資を含めていない（保守的）
3. **税金・手数料**: 実際の税引後リターンは大幅に低下

**結論**: リターンは「出来過ぎ」ではなく、特定の市場環境と戦略の組み合わせで実現可能。ただし、生存者バイアスにより5-15%程度の過大評価の可能性あり。

## 2. 疑惑ポイント: Sharpe 1.23 は信頼できるか？

### 疑惑
- Sharpe > 1.0 は「聖杯」レベル
- Walk-Forwardで劣化しているのに全期間が高いのは矛盾

### 回答・エビデンス

**Sharpe計算の検証:**
```
年率リターン: 29.64%
年率ボラティリティ: 22.66%
リスクフリーレート: 0.1% (想定)
Sharpe = (29.64% - 0.1%) / 22.66% = 1.30 (計算値)
報告値: 1.23 (保守的)
```

**Walk-Forward劣化の説明:**
| Phase | Sharpe | 期間特性 |
|-------|--------|----------|
| Training (2007-2015) | 1.12 | リーマン底値+アベノミクス |
| Validation (2016-2020) | 1.51 | 低金利・日銀緩和 |
| OOS (2021-2026) | 0.58 | 金利上昇・グロース優位 |

**全期間Sharpe > OOS Sharpeの理由:**
- Training/Validationの高いSharpeが全期間を引き上げ
- これは「過剰適合」ではなく「レジーム依存」を示す

**結論**: Sharpe 1.23は計算上正確だが、OOS期間の劣化は戦略の限界を示唆。

## 3. 疑惑ポイント: データ品質問題

### 自己申告

本バックテストには以下のデータ品質問題が存在します:

1. **2021年以降のデータカバレッジ低下**
   - 2020年: 4,020銘柄
   - 2021年: 1,003銘柄（75%減少）
   - 原因: J-Quants APIからの取り込み不完全

2. **価格欠損時のフォールバック処理**
   - 初期コード: 欠損時に `entry_price * 0.5` を使用
   - 影響: 8トレードで人工的な-50%損失が発生
   - 修正: 価格欠損時はクローズをスキップ

3. **財務データの遅延反映**
   - 決算発表日から実際のデータ反映までのラグ
   - Look-aheadバイアスの完全排除は未検証

**影響評価:**
- OOS期間のSharpeは0.33-0.58の範囲で不確実
- Training/Validation期間（2007-2020）は信頼性が高い

## 4. 疑惑ポイント: 300億円キャパシティ

### 自己申告

**バックテストの限界:**
- 初期資本: 1,000万円（300億円の0.03%）
- 最大ポジション: 約5,900万円
- 実際の300億円運用とは60倍のスケール差

**スケーラビリティ計算:**
- 現行インパクト: 28 bps
- 単純スケール時: 220 bps（破綻レベル）
- 対策後予測: 44 bps（30銘柄分散+5日執行）

**結論**: 「300億円対応可能」は条件付き主張。以下の対策が必須:
1. 30銘柄以上への分散
2. VWAP/TWAP執行
3. ADT 10億円以上へのフィルタ強化

## 5. 疑惑ポイント: 勝率66%・プロフィットファクター2.76

### 疑惑
- 勝率66%でPF 2.76は「出来過ぎ」
- 平均勝ちと平均負けのバランスは？

### エビデンス

**実際の統計:**
| 指標 | 値 |
|------|-----|
| 勝ちトレード | 145/219 (66.2%) |
| 平均勝ち | +68.9% |
| 平均負け | -27.6% |
| 勝ち/負け比率 | 2.50x |

**PF計算検証:**
```
総利益 = 145 × 68.9% × 平均ポジション
総損失 = 74 × 27.6% × 平均ポジション
PF = (145 × 68.9) / (74 × 27.6) = 4.89 (理論値)
実測値: 2.76 (乖離あり - ポジションサイズの不均一性による)
```

**正当性:**
- バリュー戦略は「大きく勝ち、小さく負ける」特性を持つ
- PBR<1.0銘柄が適正価格に回帰する際、上昇余地は大きい
- 一方、下落リスクは資産価値で下支えされる

## 6. 疑惑ポイント: 平均保有期間344日

### 疑惑
- 1年近い保有期間は現実的か
- 途中での損切りは適切か

### 回答

**設計意図:**
- バリュー投資は本質的に長期戦略
- 252営業日（1年）の保有期間は一般的

**損切りルール:**
- ストップロス: -25%で強制クローズ
- 実際の発動: 全219トレード中、多数が発動

**改善余地:**
- よりアクティブな損切り（-15%など）でDD削減可能
- ただしリターンも低下するトレードオフ
